
# Dumbways TODO App

Simple project made with react native to finish the task.
## Run Locally

Clone the project

```bash
  git clone https://github.com/suryaa6666/dw_waystodo
```

Go to the project directory

```bash
  cd dw_waystodo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
